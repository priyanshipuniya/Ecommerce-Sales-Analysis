# 🛒 E-commerce Sales Analysis

## 📌 Project Overview
This project analyzes e-commerce sales data to identify trends, top-performing products, and regional performance.

## 🛠 Tools Used
- SQL (MySQL)
- Power BI
- Excel

## 📊 Key Analysis
- Total revenue calculation
- Top-selling products
- Sales by region
- Monthly sales trends

## 📈 Insights
- Identified top-performing regions and products
- Observed seasonal trends in sales
- Highlighted areas of low profitability

## 📁 Files in Repository
- dataset.csv
- analysis.sql
- dashboard.pbix

## 🚀 Outcome
This project helps businesses make data-driven decisions to improve revenue and performance.
